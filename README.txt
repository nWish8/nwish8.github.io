# Nick Wishart's Personal Website

This repository contains the source code for my personal site. The pages are built using the Spectral theme from [HTML5 UP](https://html5up.net) with customisations for my own content.

## Usage
Open `index.html` in your browser or serve the folder with any static web server to view the site.
